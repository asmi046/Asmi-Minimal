Создай модельи:

- Portfolio
- Services

Для этих моделей создай миграции, контроллеры и сиды.

Список полей для моделей

## Portfolio

- Заголовок (строковое обязательное)
- Слаг (строковое обязательное)
- Категория (строковое обязательное)
- Порядок сортировки (Числовой по умолчанию 100)
- Обложка (строковое обязательное, длинна 800 символов)
- Описание (Текстовое поле, nullable)

## Services

- Заголовок (строковое обязательное)
- Слаг (строковое обязательное)
- Категория (строковое обязательное)
- Порядок сортировки (Числовой по умолчанию 100)

Для каждой модели сделай 2 роута основная категория и страница с слагом в виде параметра
